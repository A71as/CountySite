# Brand logo assets

Use `LOGO_ASSETS` in `src/lib/constants/images.ts`. Each asset has a specific context:

| Asset | Context |
|-------|---------|
| **Speech bubble only.png** | **Navbar** — compact center logo (bubble only). |
| **LOGO - horizontal .png** | Full lockup, horizontal layout. |
| **Logo - vertical .png** | **Hero** and **Footer** — vertical stack (Footer inverted for dark bg). |
| **LOGO - No speech bubble.png** | **OG / social** — text-only lockup for shares and previews. |
| **Brand guide.png** | Reference only (do not display as logo). |
| **BarberChop.otf** | Brand font; add via `@font-face` for campaign-art typography. |
| **NJWFP.png**, **NNJDSA.png** | Endorsement org logos (in `endorsements` data). |
